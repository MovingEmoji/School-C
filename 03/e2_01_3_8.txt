セグメンテーションフォールト
    メモリのアクセス禁止エリアにアクセスしたときや読み込み専用のエリアに書きこもうとしたときに発生する実行エラー
    要約：メモリ不足、NULLポインタへアクセス

int main() {
    char *p = NULL;
    p[0] = 'C';
    return 0
}

スタックオーバーフロー
    関数などのローカル変数はスタック領域という場所に書き込まれるため再帰関数などで毎回ローカル変数を新たに確保すると領域がたりなくなってしい発生する実行エラー
    セグメンテーションフォールトの一部

int main() {
    int arr[10000*10000];
    return 0;
}
